this page shows how to use icons in bexis 2. you will also find an overview of all icons and the usecases for them.

SetIcon

This example shows how to set a Icon.
Free Solid Awesome Font Icons v5https://www.npmjs.com/package/svelte-fa
faAngleRight - fa-angle-right faAngleDown - fa-angle-down faAddressBook - fa-adress-book
html 
<script>
	import Fa from 'svelte-fa/src/fa.svelte';

	import { faAngleDown } from '@fortawesome/free-solid-svg-icons';
	import { faAngleRight } from '@fortawesome/free-solid-svg-icons';
	import { faAddressBook } from '@fortawesome/free-solid-svg-icons';
</script>

<div class="grid grid-cols-2">
	<Fa icon={faAngleRight} />
	<span><b>faAngleRight</b> - fa-angle-right</span>

	<Fa icon={faAngleDown} />
	<span><b>faAngleDown</b> - fa-angle-down</span>

	<Fa icon={faAddressBook} />
	<span><b>faAddressBook</b> - fa-adress-book </span>
</div>
